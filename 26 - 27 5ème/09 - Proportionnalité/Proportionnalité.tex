\subsection{Reconnaître une situation de proportionnalité}
    
    \begin{propriete} 
        Dans une situation de proportionnalité, les quotients des nombres de la première ligne par ceux de la seconde sont tous égaux. Le nombre trouvé s'appelle un \textbf{coefficient de proportionnalité}.
    \end{propriete}

    \begin{enumerate}[leftmargin=*]
        \item[a)] On peut utiliser un \textbf{coefficient de proportionnalité}.
        \item[b)] On peut également raisonner en \textbf{combinant différentes colonnes} afin d'obtenir de nouvelles données.
    \end{enumerate}

\section{La notion de ratio}
    Un \textbf{ratio} sert à modéliser un partage ou un mélange inégal.

    \begin{exemple}
        Pour obtenir une grenadine, on décide de mélanger 3 doses de sirop avec 4 doses d'eau.

        \medskip

        \begin{center}
            \begin{tikzpicture}
                % Les 3 parts de Sirop (Ratio 3)
                \foreach \i in {0,1,2} {
                    \draw[thick, fill=green!60] (1.2*\i, 0) rectangle (1.2*\i + 1, 1);
                    \node at (1.2*\i + 0.5, -0.3) {\footnotesize Sirop};
                }
                
                % Les 4 parts d'Eau (Ratio 4)
                \foreach \i in {3,4,5,6} {
                    \draw[thick, fill=cyan!40] (1.2*\i, 0) rectangle (1.2*\i + 1, 1);
                    \node at (1.2*\i + 0.5, -0.3) {\footnotesize Eau};
                }
            
                % Accolade pour le total
                \draw [thick, decoration={brace, mirror, raise=0.8cm}, decorate] 
                    (0,0) -- (8.2,0) node [pos=0.5, anchor=north, yshift=-1cm] 
                    {Total = 7 parts égales};
            \end{tikzpicture}
        \end{center}

        \medskip

        Ce mélange n'est pas équitable mais il donne une recette qui permet d'obtenir une grenadine qui aura toujours le même goût tant que l'on respecte le dosage de départ : 3 doses de menthe pour 4 doses d'eau.

        \medskip

        \textbf{On dit que le sirop de grenadine et l'eau sont présents dans le mélange dans un ratio de 3:4 (on le lit 3 pour 4).}
    \end{exemple}


    \begin{definition}
        \begin{itemize}[leftmargin=*]
            \item Dire que deux quantités sont dans un ratio $3:4$ (on lit "3 pour 4"), cela signifie que pour chaque groupe de 3 parts de la première quantité, on doit ajouter 4 parts de la seconde quantité.

            \item Dire que trois quantités sont dans un ratio $2:3:7$ (on lit "2 pour 3 pour 7"), cela signifie que pour chaque groupe de 2 parts de la première quantité, on doit ajouter 3 parts de la deuxième quantité et 7 parts de la dernière.
        \end{itemize}
    \end{definition}


    \begin{exemple}
        Pour préparer une citronnade, on utilise un ratio de $3:5$ entre le jus de citron et l'eau. Cela veut dire que si je mets 3 cuillères de citron, je dois mettre 5 cuillères d'eau.

        \medskip
        
        Le mélange final est composé de 8 (3+5) parts égales.

    Pour finir, on peut donner une version plus mathématiques de ces définitions.
    \begin{definition}
        Sur des exemples :
        
        \begin{itemize}
            \item On dit que 2 nombres $a$ et $b$ sont dans le ratio $3:4$ (on lit 3 pour 4) si

            \[\dfrac{a}{3}=\dfrac{b}{4}\]

            \item On dit que 3 nombres $a, b$ et $c$ sont dans le ratio $2:3:7$ (on lit 2 pour 3 pour 7) si 

            \[\dfrac{a}{2}=\dfrac{b}{3}=\dfrac{c}{7}\]
        \end{itemize}
    \end{definition}
